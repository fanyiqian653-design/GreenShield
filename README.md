# 绿盾版本修改器

一个用于修改iOS系统版本信息的应用，支持巨魔安装。

## 功能特点

- 🎯 **自动检测系统版本**：智能识别当前iOS系统版本
- 🔧 **一键修改**：点击按钮即可修改系统版本信息
- 🔄 **恢复功能**：支持恢复到原始系统版本
- 🔒 **加密保护**：采用XOR加密算法保护关键信息
- 🛡️ **防调试**：内置防调试检测机制
- 🎨 **高级UI**：现代化界面设计，美观大方

## 使用方法

1. **构建IPA文件**：
   - 进入GitHub Actions页面
   - 运行"Build iOS App"工作流
   - 下载生成的IPA文件

2. **安装应用**：
   - 使用全能签或万能签签名并安装
   - 打开应用，点击"修改绿盾"按钮
   - 如需恢复，点击"恢复"按钮

## 技术原理

- **系统版本检测**：使用`UIDevice.current.systemVersion`获取当前系统版本
- **文件修改**：修改`/var/containers/Shared/SystemGroup/systemgroup.com.apple.mobilegestaltcache/Library/Caches/com.apple.MobileGestalt.plist`文件
- **备份机制**：在修改前自动备份原始文件
- **权限处理**：需要系统级权限才能修改文件

## 注意事项

- 此应用仅用于学习和研究目的
- 修改系统文件可能会影响设备稳定性
- 使用前请确保已备份重要数据
- 部分功能可能需要越狱环境支持
